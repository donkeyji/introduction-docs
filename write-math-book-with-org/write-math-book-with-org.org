* 为什么要用 org-mode 

好处:
    - 同时获得书籍的 pdf 版本和 html 版本，分别用于离线阅读和在线阅读.
    - org-mode 内容与格式分离的思想，一定程度上可以远离繁复的 LaTeX 排版命令.
      
要解决的问题:
    1. 公式的输入和排版，包括行内公式和行间公式，行间公式还分单行公式和多行公式.
    2. 标签与引用，即 LaTeX 的 label 与 ref.
    3. 自动编号，包括定理、公式、图表.
    4. 图片插入与引用，图片的格式.
    5. 参考文献的引用.
    6. 脚注的使用.
    7. 多文档，是否还要像 latex 一样分割为多文档工程.
    8. auctex 和 cdlatex 的使用.
